JMX基于RMI,获得客户端ip的方法是和RMI的方法一样的：

 

public String getClientIp()
	{
		try
		{
			String clienthost = RemoteServer.getClientHost();
			InetAddress ia = java.net.InetAddress.getByName(clienthost);
			return ia.getHostAddress();
		}
		catch (Exception e)
		{
			Log.error(e);
			return "Unknow Client";
		}
	}
