查看下面的代码：

<?php
class A{
	protected $b = ''; 
	
	function __set($name,$value){
		echo "$name:$value<br>";
		$this->$name = $value;
	}	
		
	function __get($name){
		return $this->$name;
	}
}

$a = new A();

$a->b = 'bv';

echo $a->b.'<br>';

?>

为类A定义了属性b,也定义了属性访问方法。

这里要注意属性$b的访问控制：

如果为private 或 proteced,那么对b属性的访问是通过__set方法的。

如果为public,那么对属性b的访问没有通过__set和__get而是直接访问了。
